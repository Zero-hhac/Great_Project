# ✦ 个人博客 (Go-Blog)

一个基于 Go 语言和 Gin 框架开发的现代化个人博客系统，具有精美的 UI 设计和完整的后台管理功能。

## 🌟 项目亮点与实现效果

### 1. 现代化 UI/UX 设计
- **响应式布局**：采用宽屏双栏设计，在大屏幕上提供极佳的视觉体验。
- **毛玻璃效果**：导航栏与卡片采用半透明模糊效果，提升页面质感。
- **个人名片**：左侧固定展示个人资料、技能栈（Skill Tag）以及社交媒体链接。

### 2. 核心功能
- **文章管理 (CRUD)**：支持文章的发布、列表展示、详细阅读、编辑和删除。
- **管理权限控制**：
  - `admin` 用户拥有全局管理权限（修改个人信息、删除任何文章/评论、管理照片墙）。
  - 普通用户可以注册登录，发布文章，并管理自己发布的内容。
- **互动评论系统**：文章详情页支持发表评论，采用框式气泡布局，显示评论者昵称及发布时间。
- **首页精彩瞬间 (Gallery)**：动态照片墙展示，支持管理员在线添加、修改和删除照片及其描述。
- **数据库驱动**：所有核心数据（文章、用户、评论、个人资料、照片）均已实现数据库持久化。

### 3. 云端适配
- **环境变量驱动**：支持通过 `.env` 或系统环境变量动态配置数据库连接。
- **多数据库支持**：完美适配 PostgreSQL (Neon) 和 MySQL，支持通过 `DATABASE_URL` 自动识别。
- **Docker 部署**：提供多阶段构建的 Dockerfile，支持在 Render 等平台一键部署。

## 🛠️ 技术栈

### 后端 (Backend)
- **语言**：[Go (Golang)](https://golang.org/) 1.26+
- **框架**：[Gin Web Framework](https://github.com/gin-gonic/gin)
- **数据库 ORM**：[GORM](https://gorm.io/)
- **数据库支持**：PostgreSQL (Neon), MySQL, SQLite
- **安全**：Bcrypt 密码加密，基于 Cookie 的会话管理
- **依赖管理**：Go Modules, godotenv

### 前端 (Frontend)
- **模板引擎**：Go 标准库 `html/template`
- **样式**：原生 CSS3（使用 CSS Variables, Flexbox, Grid 布局）
- **图标**：Emoji & 自定义矢量图形

### 部署与运维
- **部署平台**：Render / Cloudflare
- **容器化**：Docker (Alpine Linux)
- **域名管理**：阿里云解析 + Cloudflare DNS
